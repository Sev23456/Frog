# =============================================================================
# 🧠 АРХИТЕКТУРА ИНТЕГРАЦИИ
# =============================================================================

"""
BioFrogAgent объединяет все компоненты в этой последовательности:

1️⃣ СЕНСОРНЫЙ ВВОД
   visual_scene (800×600) 
        ↓
   RetinalProcessing (10×10 фильтров ON/OFF)
        ↓
   retinal_output (100 нейронов)

2️⃣ ЦЕНТРАЛЬНАЯ ОБРАБОТКА
   retinal_output + motion_vectors
        ↓
   Tectum (16 колонок, направленная селективность)
        ↓
   tectal_output + movement_command

3️⃣ МОТОРНЫЙ ВЫХОД
   movement_command
        ↓
   MotorHierarchy (executive → coordination → motor)
        ↓
   muscle_activation (скорость движения)

4️⃣ НЕЙРОМОДУЛЯЦИЯ
   reward_signal → dopamine (+0.3)
   neural_activity → serotonin
   energy_level → fatigue
   
   В режиме детства:
   - dopamine: 0.85 (вместо 0.5)
   - serotonin: 0.75 (вместо 0.5)
   
5️⃣ МЕТАБОЛИЗМ
   movement_intensity + neural_activity
        ↓
   SystemicMetabolism (glucose, oxygen, fatigue)
        ↓
   energy_level

6️⃣ ПЛАСТИЧНОСТЬ
   neural_activity
        ↓
   FunctionalPlasticityManager (гомеостаз)
   StructuralPlasticityManager (создание/удаление синапсов)
        ↓
   обновлённые synaptic_weights
"""


# =============================================================================
# 👶 РЕЖИМ "ДЕТСТВА" (JUVENILE MODE)

"""
Детство - это критически важный период развития, где:

✓ ПОВЫШЕННАЯ НЕЙРОМОДУЛЯЦИЯ:
  - Дофамин: 0.85 → стимулирует исследование и обучение
  - Серотонин: 0.75 → улучшает настроение и обучение
  - Норм. взросле: dopamine=0.5, serotonin=0.5

✓ ОБЛЕГЧЁННАЯ ОХОТА:
  - Радиус поимки: 100 пикселей (вместо 20)
  - Вероятность успеха: 60% (вместо 100%)
  - Это даёт лягушке шанс получить первые успехи

✓ ДЛИТЕЛЬНОСТЬ:
  - juvenile_duration = 5000 шагов
  - После этого лягушка становится взрослой

✓ БИОЛОГИЧЕСКОЕ ОБОСНОВАНИЕ:
  - В реальности молодые животные имеют повышенный дофамин
  - Это стимулирует игру и исследование
  - Помогает быстро обучиться критическим навыкам

ИСПОЛЬЗОВАНИЕ:

    sim = BioFlyCatchingSimulation(juvenile_mode=True)
    
    # Без режима детства (сразу взрослая)

"""


# =============================================================================
# 📊 ПАРАМЕТРЫ И КОНСТАНТ
# =============================================================================

"""
НЕЙРОНЫ:
  - LIFNeuron: tau_membrane=20ms, threshold=-40mV
  - PyramidalNeuron: дендритная интеграция, апикальное усиление
  - FastSpikingInterneuron: tau=5ms, адаптация к спайкам

СИНАПСЫ:
  - STDP окно: ±50 ms
  - STDP амплитуда: 0.01
  - Модуляция: dopamine & serotonin
  - STP: facilitation (50ms), depression (200ms)

ГЛИЯ:
  - Количество: 24 астроцита в сетке 4×6
  - Кальциевая динамика: tau=500ms
  - Влияние: radius=50.0 пикселей

МЕТАБОЛИЗМ:
  - Потребление глюкозы: 0.001/s
  - Потребление кислорода: 0.0015/s
  - Циркадный цикл: ~2.8 часов (в симуляции: 10000 шагов)
  - ATP потребление на спайк: 0.1 единиц

РАЗРАБОТКА:
  - Рефрактерный период: 1-2 ms
  - Дельта-времени: 0.01 ms
  - Целевой FPS: 30-60

ДИФФУЗИЯ:
  - Разрешение сетки: 20×20
  - Гауссов ядро для диффузии
  - Exponential decay: tau=500ms

ТЕКТУМ:
  - Количество колонок: 16
  - Покрытие: 0° до 360° (22.5° шаг)
  - Нейроны на колонку: 14 (8 pyr + 4 intern + 2 output)
  - Селективность: σ=0.5 rad

МОТОРНАЯ ИЕРАРХИЯ:
  - Слой 1 (executive): 4 нейрона (UP, DOWN, LEFT, RIGHT)
  - Слой 2 (coordination): 8 интернейронов
  - Слой 3 (motor): 12 нейронов управления мышцами

"""
BioFrogAgent:
  __init__(space, position, bio_mode=True, juvenile_mode=True, ...)
    Инициализировать агента с полной биологической сетью
  
  update(dt, flies) → Dict[str, Any]
    Один цикл обновления мозга и тела
    Возвращает: position, velocity, energy, caught_flies, dopamine, ...
  
  detect_flies(flies) → (visual_scene, motion_vectors)
    Обнаружить мух в визуальном поле
  
  extend_tongue(target_position)
    Выпустить язык
  
  retract_tongue()
    Втянуть язык

BioFrogBrain:
  process_sensory_input(visual_scene) → Dict
    Обработка визуального входа через сетчатку
  
  process_motion(retinal_input, motion_vectors) → Dict
    Обработка движения через тектум
  
  generate_motor_output(movement_command) → Dict
    Генерирование моторного выхода
  
  update_neuromodulation(reward, neural_activity, dt)
    Обновление дофамина и серотонина
  
  apply_plasticity(neural_activity)
    Применение пластичности синапсов
  
  update_metabolism(movement_intensity, neural_activity, dt)
    Обновление энергии и метаболизма

BioFlyCatchingSimulation:
  __init__(width, height, bio_mode=True, juvenile_mode=True, ...)
    Создать симуляцию
  
  run_simulation(max_steps)
    Запустить симуляцию на определённое количество шагов
  
  plot_results()
    Построить графики результатов
  
  get_statistics() → Dict
    Получить статистику
  
  reset_simulation()
    Сбросить к начальному состоянию
  
  close()
    Закрыть симуляцию
"""
