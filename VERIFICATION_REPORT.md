## 🚨 КРИТИЧЕСКИЕ ПРОБЛЕМЫ

### 1. **AttributeError: 'FunctionalPlasticityManager' object has no attribute 'update'**
**Файл:** `bio_frog_agent.py:229`
**Решение:** Добавить метод `update()` или заменить вызов на `apply_homeostatic_scaling()`.

### 2. **STDP не применяется**
Синапсы созданы, но `apply_stdp()` никогда не вызывается в цикле обновления.
**Решение:** Добавить вызов в `apply_plasticity()`.

### 3. **Астроциты не получают данные**
`glial_network.update({}, [], dt)` передаёт пустые списки.
**Решение:** Передавать реальную активность нейронов.

### 4. **Неверное количество астроцитов**
Заявлено 24, создаётся 16 (4×4 grid).
**Решение:** Использовать не-square grid или исправить формулу.

### 5. **Дублирование кода пластичности**
`structural_plasticity.py` — почти точная копия части `functional_plasticity.py`.
