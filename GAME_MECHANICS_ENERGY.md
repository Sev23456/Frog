# 🎮 Игровая механика энергии в BioFrog

## ✅ Реализовано: Двойная система энергии

BioFrog теперь имеет **ДВЕ НЕЗАВИСИМЫЕ** энергетические системы:

### 1️⃣ БИОЛОГИЧЕСКАЯ ЭНЕРГИЯ (0-1 шкала)
- **Диапазон**: 0.0 - 1.0
- **Назначение**: Модулирует поведение через глиальную сеть
- **Эффекты**:
  - Влияет на скорость движения (0.3-1.0× от базовой)
  - Модулирует успех охоты (20-100% в зависимости от энергии)
  - Модулирует возбудимость сенсорного входа (0.5-1.0)
  - Модулирует нейромодуляторы через глию (дофамин/серотонин)
- **Динамика**:
  - Затрата на охоту: 0.2 энергии/сек
  - Затрата на движение: 0.005 × dt × (1 + интенсивность)
  - Пассивное восстановление: 0.0015 энергии/сек
  - Награда за поимку: +0.2 энергии

### 2️⃣ ИГРОВАЯ МЕХАНИКА ЭНЕРГИИ (0-30 шкала, как ANN/SNN)
- **Диапазон**: 0.0 - 30.0 очков
- **Назначение**: Score tracking compatible с ANN/SNN для справедливого сравнения
- **Механика** (идентична ANN):
  - Затрата на движение: **(0.08 + 0.02 × ||velocity||) × dt**
  - Награда за поимку: **+5.0** очков
  - БЕЗ пассивного восстановления
  - Фиксированный максимум: 30.0
- **Использование**:
  - `result['game_energy']` — текущее значение энергии
  - `result['game_energy_ratio']` — нормализованное (0-1) для нейросетей

## 📍 Локация в коде

### Инициализация (`__init__`)
```python
# Биологическая энергия
self.energy = 1.0

# Игровая механика (параллельная система)
self.game_energy = 30.0
self.max_game_energy = 30.0
```

### Обновление при поимке (`update()`)
```python
if self.attached_fly:
    # Биологическая награда
    self.energy = min(1.0, self.energy + 0.2)
    
    # Игровая награда (ANN/SNN совместимость)
    self.game_energy = min(self.max_game_energy, self.game_energy + 5.0)
```

### Затраты при движении (`update()`)
```python
# Биологическая затрата
energy_cost = 0.005 * dt * (1.0 + movement_intensity)
self.energy = max(0.0, self.energy - energy_cost)

# Игровая затрата (как ANN)
game_energy_cost = (0.08 + 0.02 * last_velocity_norm) * dt
self.game_energy = max(0.0, self.game_energy - game_energy_cost)
```

### Пассивное восстановление (только биология)
```python
resting_recovery = 0.0015 * dt
self.energy = min(1.0, self.energy + resting_recovery)
# game_energy НЕ восстанавливается (как в ANN/SNN)
```

## 🎯 Назначение

- **Биологическая энергия** остаётся полноценной системой:
  - Модулирует поведение через глиальную сеть
  - Имеет пассивное восстановление (реалистично)
  - Дорогая охота (0.2/сек)

- **Игровая энергия** для справедливого сравнения:
  - Идентична ANN/SNN механике
  - Позволяет сравнивать эффективность агентов на одних и тех же метриках
  - Может использоваться как функция вознаграждения для обучения
