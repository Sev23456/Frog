# BioFrog v2.0 - Биологически достоверный агент лягушки

## 📋 Описание

BioFrog v2.0 — это полностью переписанная с нуля биологически достоверная модель агента лягушки, основанная на спецификации из `ARCHITECTURE_COMPLETE.md`.

## 🔑 Ключевые особенности

### 1. Двойная система энергии
- **Биологическая энергия (0-1)**: Реалистичная динамика метаболизма (глюкоза, кислород, лактат)
- **Игровая энергия (0-30)**: Совместимость с ANN/SNN агентами для честного сравнения

### 2. Энергетическая модуляция ВСЕГО
Энергия влияет на:
- ✅ Сенсорное восприятие (видимость мух)
- ✅ Моторную функцию (скорость движения)
- ✅ Успех охоты (вероятность поимки)
- ✅ Нейромодуляцию (дофамин, серотонин)
- ✅ Глиальную поддержку мозга

### 3. Глия как энергетический интегратор
Глиальная сеть (25 астроцитов):
- Считывает уровень энергии от метаболизма
- Модулирует возбудимость нейронов
- Регулирует выделение глиотрансмиттеров
- Подавляет синапсы при дефиците энергии

### 4. Жизненный цикл: Детство → Взрослость
**Детство (0-5000 шагов):**
- Дофамин: 0.85 (высокое исследование)
- Серотонин: 0.75 (хорошее настроение)
- Быстрое обучение и пластичность

**Взрослость (5000+ шагов):**
- Дофамин: 0.5 (нормальный уровень)
- Серотонин: 0.5 (нормальный уровень)
- Стабильное применение выученного

## 🏗️ Архитектура

```
ОКРУЖЕНИЕ (мухи)
    ↓
СЕНСОРНАЯ СИСТЕМА (16-канальное зрение)
    ├─→ Энергетическая модуляция (0.5-1.0 фактор)
    └─→ retinal_output [16]
    
ТЕКТУМ
    ├─→ tectal_output [16]
    └─→ movement_command [2]
    
НЕЙРОМОДУЛЯЦИЯ
    ├─→ dopamine_level (0.5-0.85)
    ├─→ serotonin_level (0.5-0.75)
    └─→ Модуляция энергией
    
ГЛИЯ
    ├─→ Читает: tectal_activity, energy_level
    ├─→ Выпускает: gliotransmitter
    └─→ Модулирует: синапсы
    
МЕТАБОЛИЗМ
    ├─→ glucose_level (биологическая энергия 0-1)
    └─→ Связан: с двигательной активностью
    
ИГРОВАЯ ЭНЕРГИЯ
    ├─→ game_energy (0-30)
    ├─→ Затрачивается: на движение
    ├─→ Восполняется: поимкой мух (+5)
    └─→ Модулирует: всё поведение
    
ЯЗЫК & ОХОТА
    ├─→ Затрачивает: энергию
    ├─→ Генерирует: дофамин при награде
    └─→ Модулируется: энергией (успех 30-100%)
    
МОТОРНАЯ ИЕРАРХИЯ
    ├─→ Читает: movement_command, energy_ratio
    └─→ Выдает: velocity → физика (Pymunk)
```

## 🚀 Быстрый старт

### Пример 1: Простая симуляция

```python
from biofrog_v2 import run_simulation

# Запуск 10-секундной симуляции в режиме детства
results = run_simulation(
    duration=10.0,
    dt=0.01,
    juvenile_mode=True,
    render=False
)

print(f"Поймано мух: {results['total_caught']}")
print(f"Средняя энергия: {results['avg_energy']:.2f}")
```

### Пример 2: Ручное управление

```python
import pymunk
from biofrog_v2.bio_frog_agent import BioFrogAgent

# Создать пространство
space = pymunk.Space()
space.gravity = (0, 0)

# Создать лягушку
frog = BioFrogAgent(
    space=space,
    position=(400, 300),
    juvenile_mode=True,
    training_mode=True
)

# Создать мух
flies = [...]  # Список объектов Fly

# Цикл симуляции
dt = 0.01
for step in range(1000):
    space.step(dt)
    
    # Обновить лягушку
    state = frog.update(dt, flies)
    
    # Доступна статистика
    print(f"Шаг {step}: поймано={state['caught_flies']}, "
          f="энергия={state['game_energy']:.1f}, "
          f="дофамин={state['dopamine']:.2f}")

# Уборка
frog.remove()
```

## ⚙️ Энергетические константы

### Биологическая энергия (0-1)
- Начальное значение: 1.0
- Награда за поимку: +0.2
- Расход на движение: 0.005 × dt × (1 + intensity)
- Расход на охоту: 0.2 × dt
- Регенерация в покое: 0.0015 × dt

### Игровая энергия (0-30)
- Начальное значение: 30.0
- Награда за поимку: +5.0
- Расход на движение: (0.08 + 0.02 × velocity) × dt
- Game Over при: energy ≤ 0

## 🧠 Нейромодуляторы

### Дофамин
- Детство: 0.85 (база)
- Взрослость: 0.5 (база)
- Награда: +0.3 × reward
- Модуляция энергией: × max(0.3, glucose_level)

### Серотонин
- Детство: 0.75 (база)
- Взрослость: 0.5 (база)
- Зависит от энергии: +0.1 × glucose / 2
- Глиальная модуляция: × excitability_modulation
