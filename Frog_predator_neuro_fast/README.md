# 🐸 Frog Predator Neuro - Biorealistic Frog Brain

Полнофункциональная нейробиологическая симуляция хищной лягушки, адаптированная из архитектуры волка `Wolf_for_laba_v3_re`.

## 📋 Характеристики

### 🧠 Биологические компоненты
- **Визуальная система** (сетчатка с ON/OFF фильтрами)
- **Тектум** (обработка салиентности и движения)
- **Пространственная память** (place cells + head direction cells)
- **Базальные ганглии** (выбор действия через прямой/непрямой пути)
- **Моторная иерархия** (иерархический двигательный контроль)
- **Нейротрансмиттеры** (дофамин, серотонин, ацетилхолин)
- **Глиальная сеть** (модуляция нейронной активности)
- **Метаболизм нейронов** (ATP, энергия)
- **Пластичность** (структурная и функциональная)

### 🐸 Физиология лягушки (адаптированная)
- **Размер**: 9px радиус (крупная лягушка, 75% от размера волка)
- **Скорость**: 65 px/s максимум (прыги быстрые, но не долгие)
- **Энергия**: Максимум 0.8 (меньше чем волк)
- **Зрение**: 1.8 tile радиус (хорошее видение для охоты)
- **Мозг**: Меньший (0.7x от Wolf) - меньше place cells, колонн тектума

### 🎯 Поведение
- **Охота** (основное поведение) - отслеживание и ловля насекомых
- **Исследование** (экспоненциальная любопытство)
- **Пространственная память** - запоминание хороших мест для охоты
- **Процесс обучения** - ювенальный период с ускоренной пластичностью

### 🚫 Что убрано
- **Социальное взаимодействие** - фроги солитарные
- **Коммуникация** - нет сигналов между агентами
- **Обмен энергией** - нет группового поведения
- **Глиальная сеть для социала** - адаптирована только для внутримозговой модуляции

## 🚀 Использование

### Запуск существующего интерфейса
```bash
python -m Frog_predator_neuro.game_
```

### Быстрая проверка поведения
```bash
python -m Frog_predator_neuro.test_frog_predator
```

### Экспорт и анализ

```python
from Frog_predator_neuro.simulation import Simulation
from Frog_predator_neuro.agent import BioFrogAgent

# Создать симуляцию
sim = Simulation(headless=True)
agent = sim.agents[0]

# Запустить 30 секунд
for _ in range(3000):
    sim.update(0.01)

# Проверить результаты
print(f"Food eaten: {agent.food_eaten}")
print(f"Energy: {agent.body_energy}")
print(f"Distance traveled: {agent.total_distance_traveled}")
```

## 📊 Внутренние системы

### Аффективное состояние (без социала)
- `arousal` - возбуждение
- `curiosity` - любопытство (исследование)
- `frustration` - фрустрация
- `reward_confidence` - уверенность в награде
- `restlessness` - беспокойство

### Сенсорная система
- ON/OFF фильтры сетчатки (8 направлений)
- Детектор движения (тектум)
- Пространственная карта места (hippocampus-like)

### Моторная система
- Иерархия движений (исследование > охота > бегство)
- Инерция движения (память о предыдущем направлении)
- Адаптивное проецирование на открытые пути

## 🐸 Примечания

- Фроги **НЕ видят** друг друга (солитарные)
- Фроги **НЕ слышит** звуков других
- Фроги фокусируется только на **охоте и выживании**
- Каждый фрог имеет собственную пространственную память
- Нейробиология остаётся **реалистично близкой к биологии** 🧠
