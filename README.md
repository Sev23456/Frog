# BioFrog v2.0 — Биологически правдоподобный агент для изучения «существенности» поведения

> **Цель проекта**: Не копирование биологии ради точности, а извлечение *вычислительных принципов*, создающих поведение с признаками «существенности» — адаптивности, усталости, развития, мотивации — в отличие от традиционных ИИ как «интеллектуальных калькуляторов».

---

## Философия проекта

| Традиционный агент (DQN/PPO) | BioFrog v2.0 | Живое существо |
|------------------------------|--------------|----------------|
| ❌ Постоянная активность | ✅ Циклы активности/усталости | ✅ Циркадные ритмы |
| ❌ Обучение = глобальный градиент | ✅ Локальные правила + нейромодуляция | ✅ Синаптическая пластичность |
| ❌ Отсутствие развития | ✅ Режим «детства» → взросление | ✅ Критические периоды |
| ❌ Абстрактная функция | ✅ Эмерджентное поведение | ✅ Адаптивность среде |

**Ключевой инсайт**: «Существенность» возникает не из анатомической точности, а из *правильного взаимодействия* нескольких механизмов:
- пластичность (кратко- и долгосрочная)
- нейромодуляция (дофамин, серотонин)
- метаболизм (энергия → поведение)
- иерархическая архитектура (сенсор → моторика)

---

## Интегрированные биологические механизмы

### 1. Нейронное разнообразие
- **LIF-нейроны**: базовая интеграция
- **Пирамидальные нейроны**: нелинейная дендритная интеграция (апикальный/базальный вход)
- **Быстро спайкающие интернейроны**: локальное торможение с адаптацией

### 2. Синаптическая пластичность
- **STDP** (Spike-Timing-Dependent Plasticity): обучение причинно-следственным связям
- **STP** (Short-Term Plasticity): адаптация к фоновой активности (фасилитация/депрессия)
- **Нейромодуляция пластичности**: дофамин как «гейт» обучения

### 3. Сенсомоторная архитектура
```
Сетчатка (ON/OFF фильтры)
       ↓
Тектум (16 колонок, направленная селективность)
       ↓
Моторная иерархия (исполнительный → координационный → мышечный слои)
```

### 4. Метаболическая система
- Глюкоза/кислород → энергия нейронов → возбудимость
- Циркадный ритм → естественные циклы активности/усталости
- Лактат → утомление при интенсивной активности

### 5. Нейромодуляторы
- **Дофамин**: сигнал награды → усиление пластичности
- **Серотонин**: регуляция насыщения и устойчивости
- **Ацетилхолин**: внимание и новизна
- *Пространственная диффузия* через 2D-сетку (не только скаляр)

### 6. Глиальная модуляция
- Астроциты реагируют на локальную активность → кальциевые всплески
- Выброс глиотрансмиттеров (глутамат, АТФ) → модуляция синапсов

## Режимы работы:

1. **Детство** — повышенный дофамин, широкий радиус поимки (обучение)
2. **Взросление** — нормальный дофамин, точная охота (эксплуатация)
3. **Headless** — быстрая симуляция без визуализации
4. **Сравнение** — детство vs. взросление

## 📊 Что можно наблюдать

| Явление | Как проявляется | Биологический аналог |
|---------|-----------------|----------------------|
| **Адаптация к фону** | Агент перестаёт реагировать на статичные объекты | Сенсорная адаптация сетчатки |
| **Развитие** | Детская лягушка → взрослая: снижение дофамина, повышение точности | Критический период развития |
| **Усталость** | При низкой энергии — меньше движения, больше отдыха | Циркадные ритмы амфибий |
| **Обучение** | Успешные поимки → дофаминовый выброс → усиление синапсов | Обучение с подкреплением |
| **Внимание** | Фокусировка на движущихся объектах, игнорирование фона | Селективное внимание тектума |

## Осознанные компромиссы

Проект **не претендует** на нейрофизиологическую точность. Упрощения сделаны *сознательно* для игровой среды:

| Компонент | Упрощение | Почему допустимо |
|-----------|-----------|------------------|
| Пирамидальные нейроны в тектуме | Анатомически неточны (у лягушек их нет) | Сохраняется *функция* (нелинейная интеграция), а не морфология |
| Биомеханика языка | Линейная модель вместо эластичной | В 2D-среде кинематика не влияет на поведенческую динамику |
| Детектор размера объекта | Отсутствует (реакция на любое движение) | ⚠️ **Единственное критическое упущение** |
| Диффузия нейромодуляторов | Низкое разрешение сетки (20×20) | Сохраняется *временная динамика*, пространственная точность вторична |

## 📚 Научная основа

- **STDP**: Bi & Poo (1998), Markram et al. (1997)
- **Тектум лягушки**: Ewert (1987) — детектор «маленький тёмный объект в движении»
- **Нейромодуляция пластичности**: Reynolds & Wickens (2002) — дофамин как гейт обучения
- **Глиальная модуляция**: Araque et al. (2014) — тричастичный синапс (нейрон-астроцит-нейрон)

## 💡 Заключение

BioFrog v2.0 — это **поведенческая лаборатория**, а не биомиметический робот. Проект демонстрирует, что «существенность» поведения возникает из *взаимодействия* простых механизмов:

> **Пластичность + Нейромодуляция + Метаболизм = Адаптивное поведение**

Добавление одного фильтра размера объектов превратит агента из «общего детектора движения» в хищника с экологической нишей — и это минимальный порог для проявления «существенности».
