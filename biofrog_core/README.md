# BioFrog Core v3.0

Чистая биологическая модель лягушки, переписанная с нуля на основе `ARCHITECTURE_COMPLETE.md`.

## 🚀 Особенности

### 1. Двойная система энергии
- **Биологическая (0-1)**: медленная динамика от метаболизма (глюкоза, кислород, лактат)
- **Игровая (0-30)**: быстрая динамика для совместимости с ANN/SNN

### 2. Энергетическая модуляция ВСЕГО
Энергия влияет на:
- ✅ Сенсорику: видимость × (0.5–1.0)
- ✅ Моторику: скорость × max(0.3, energy)
- ✅ Охоту: успех × max(0.3, energy)
- ✅ Нейромодуляцию: дофамин/серотонин
- ✅ Глию: поддержка синапсов

### 3. Реалистичный метаболизм
- Аэробный/анаэробный режим
- Гликоген как запас энергии
- Лактат как продукт усталости
- Медленное восстановление в покое

### 4. Жизненный цикл
- **Детство** (0–5000 шагов): дофамин 0.85, серотонин 0.75, высокая пластичность
- **Взрослость** (5000+ шагов): дофамин 0.5, серотонин 0.5, стабильность

## 🔧 Использование

### Базовый пример

```python
from bio_frog import BioFrogAgent

# Создание агента
frog = BioFrogAgent(initial_position=(400, 300), juvenile_mode=True)

# Мухи на сцене
flies = [
    {'x': 450, 'y': 320, 'brightness': 1.0},
    {'x': 380, 'y': 280, 'brightness': 0.8},
]

# Симуляция
dt = 0.01  # 10 мс
for step in range(1000):
    result = frog.update(flies=flies, dt=dt)
    
    if result['caught_fly']:
        print(f"Поймал муху! Энергия: {result['game_energy']:.1f}")
    
    if result['game_over']:
        print("GAME OVER!")
        break
```

### Доступные состояния

```python
result = frog.update(flies, dt)

# Позиция и движение
result['position']      # [x, y]
result['velocity']      # [vx, vy]

# Энергия
result['bio_energy']    # 0-1, биологическая
result['game_energy']   # 0-30, игровая

# Нейромодуляторы
result['dopamine']      # 0-1
result['serotonin']     # 0-1

# Охота
result['caught_fly']    # bool
result['caught_flies']  # всего поймано
result['tongue_extended']  # bool

# Мозг
result['tectal_output']     # [16] активность тектума
result['gliotransmitter']   # уровень глиотрансмиттера
result['energy_stress']     # bool, есть ли дефицит энергии

# Жизненный цикл
result['is_juvenile']   # bool
result['juvenile_age']  # шаг с начала детства
```

## 📊 Архитектурные принципы

### 1. Энергия повсюду
❌ **НЕПРАВИЛЬНО**: Энергия влияет только на скорость

✅ **ПРАВИЛЬНО**: Энергия модулирует:
- Сенсорное восприятие (что мозг видит)
- Успех охоты (вероятность поймать)
- Скорость движения (как быстро двигаться)
- Нейромодуляцию (дофамин/серотонин)
- Глиальную работу (метаболическую поддержку мозга)

### 2. Двухуровневая обработка
У BioFrog ДОЛЖНЫ быть две энергии:
1. **Биологическая (0-1)** — для реалистичной динамики тела
2. **Игровая (0-30)** — для совместимости с ANN/SNN

### 3. Глия — главный модулятор
Глия не просто выделяет вещества, она:
- Считывает энергию от метаболизма
- Проверяет может ли мозг позволить себе текущий уровень активности
- Подавляет синапсы если энергии не хватает
- Усиливает синапсы если есть избыток

### 4. Информационные потоки

```
energy → сенсорика → что видит мозг
energy → моторика → как быстро двигаться
energy → нейромодуляция → настроение мозга
energy → глия → поддержка мозга
energy → охота → вероятность поимки
```

Всё должно быть СВЯЗАНО через энергию!
